## ⚙️ 운영 방식

```
각자 브랜치에서 작업 후 Main으로 PR 날리기
```

1. `main` 브랜치에서 **본인 이름으로 브랜치** 생성
   ```bash
   git checkout -b 김가원
   ```
2. 본인 폴더 안에 풀이 파일 추가
3. 커밋 & 푸시 후 **`main`으로 PR** 생성
4. 최소 1명 이상의 **Approve** 후 머지
